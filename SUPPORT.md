# Support

Language: English | [Simplified Chinese](SUPPORT.zh-CN.md)

For usage questions, start with `README.md`.

When opening an issue, include:

- Windows version.
- PowerShell version: `$PSVersionTable.PSVersion`.
- Whether you used bundled Node.js or `-UseSystemNode`.
- Whether `vendor/manifest.json` and `vendor/checksums.sha256` exist.
- The exact command you ran, with secrets removed.
- Error output, with secrets removed.

For security-sensitive issues, see `SECURITY.md`.
