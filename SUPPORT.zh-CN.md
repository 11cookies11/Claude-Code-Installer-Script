# 支持

语言：简体中文 | [English](SUPPORT.md)

使用问题请先查看 `README.zh-CN.md`。

提交 Issue 时建议包含：

- Windows 版本。
- PowerShell 版本：`$PSVersionTable.PSVersion`。
- 使用的是内置 Node.js 还是 `-UseSystemNode`。
- `vendor/manifest.json` 和 `vendor/checksums.sha256` 是否存在。
- 实际执行的命令，注意移除密钥。
- 错误输出，注意移除密钥。

安全敏感问题请查看 `SECURITY.zh-CN.md`。
