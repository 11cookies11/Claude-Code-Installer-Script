# Claude Code Release Bundle

This folder ships with the release archive together with the root
`install-claude-code-deepseek.ps1` script.

## How to run

From the extracted release root:

```powershell
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
.\install-claude-code-deepseek.ps1
```

The installer automatically looks for `src\vendor`, so you do not need to
pass an offline resource path.

## Bundle layout

- `install-claude-code-deepseek.ps1`
- `src\README.md`
- `src\README.zh-CN.md`
- `src\vendor\`

The `vendor` folder is regenerated by the release workflow so the archive
always carries fresh offline resources.

## Interactive flow

The wizard asks for the DeepSeek token first, then lets you choose:

1. Permission level
2. Conversation entry

You can also automate the script with `-NonInteractive`.
